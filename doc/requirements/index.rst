Setup Requirement
-----------------

.. req:: syspilot is installed and operational
   :id: REQ_SYSPILOT_001
   :status: done
   :tags: meta, tooling
   :satisfies: SPEC_ARCH_BUILD

   The syspilot documentation framework shall be installed and produce
   a valid Sphinx build without errors.  This is a meta-requirement covering
   the documentation toolchain itself, satisfied by the build/CI architecture.
